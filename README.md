# homebrew-gallery

Homebrew tap for Gallery App.

## Install

```bash
brew tap iddqd-hr/gallery https://iddqd-hr@dev.azure.com/iddqd-hr/claudegallery/_git/homebrew-gallery
brew install --cask gallery-app
```

## Update

```bash
brew upgrade gallery-app
```

## Uninstall

```bash
brew uninstall gallery-app
```
